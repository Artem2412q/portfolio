# Portfolio Card — v11 (premium minimal)

## Что внутри
- Полный редизайн: “дорого, минималистично”, с нормальной сеткой и будущей расширяемостью
- Светлая тема (серо‑бежевая) + тёмная тема (аналогия)
- Новый фон: мягкие волны + зерно, без трекинга мыши
- Переработанные анимации: reveal on scroll, микро‑UX, прогресс скролла
- Закреплённая верхняя панель (sticky)
- Портфолио: кейсы с раскрытием (UX / Решение / Результат) через модалку
- Подсказки (tooltips) и эмодзи — аккуратно и по делу
- Контактная форма с отправкой на `artemcepko69@gmail.com` через FormSubmit
- Антиспам-капча (локальная математика) + место под Turnstile/hCaptcha (опционально)

## Важно про отправку писем
Используется FormSubmit:
- action: `https://formsubmit.co/artemcepko69@gmail.com`
- Первый раз FormSubmit может попросить подтверждение почты у владельца.
- Параметры `_next` и `_subject` можно менять в `index.html`.

## CAPTCHA (варианты)
1) Уже работает локальная “антиспам” математика (без ключей и бэкенда).
2) Если хочешь настоящую капчу:
   - Cloudflare Turnstile или hCaptcha
   - Вставь их виджет в блок `<div class="captchaExt" id="captchaExt">...`
   - В `app.js` добавь проверку наличия токена перед отправкой.

## Где менять тексты/кейсы
`app.js` → массив `CASES` (описания, теги, изображения, ссылки).
